System Overview

What it does

Continuous purple-team validation: a custom Go agent runs MITRE ATT&CK-mapped tests on endpoints, results stream into Elasticsearch as per-control documents, and the Analytics layer correlates execution against Microsoft Defender's alerts and Secure Score to produce a measured Defense Score over time.

Why two backends?

backend/ targets always-on hosts (Docker, Render, Fly, Railway) with sync SQLite and in-process schedulers. backend-serverless/ targets Vercel — same Express app, but Turso async DB, Vercel Blob, Cron-driven schedulers, and a /api/capabilities endpoint that tells the frontend which features to hide (Go builds, runtime git sync, etc.).
