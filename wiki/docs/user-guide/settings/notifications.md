# Notifications

## Notification Bell

The notification bell in the top navigation bar shows recent threshold breach alerts. Click the bell to see:

- Alert type (Defense Score drop, absolute floor breach)
- Timestamp
- Affected metric and current value
- Link to the relevant Analytics view

## Notification History

The bell dropdown shows the most recent alerts. Older alerts are available in the Settings → Alerting section.

See [Alerting (Slack & Email)](../integrations/alerting) for configuring external notification channels.
