# Visual Themes

## Available Themes

### Default
Clean light/dark mode with a purple accent color. Professional appearance suitable for most environments.

### Neobrutalism
Bold design with a hot pink accent color, thick borders, and high contrast. Designed for a playful, modern aesthetic.

### Hacker Terminal
Retro terminal aesthetic with phosphor green or amber text on a dark background. Includes scanline effects for the full "Hollywood hacker" experience.

## Switching Themes

1. Navigate to **Settings**
2. Open the **Theme** selector
3. Choose your preferred theme
4. The change applies immediately across all pages

## Technical Details

Themes are implemented using CSS variables in Tailwind CSS v4 `@theme` blocks. Each theme overrides the base color palette, border styles, and special effects (like scanlines for Hacker Terminal).
